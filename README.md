# ZenByteLabs | Premium Portfolio

A high-fidelity, modern digital portfolio for **ZenByteLabs**, built with React, Vite, and Framer Motion. This project showcases premium design aesthetics, clinical engineering, and high-performance animations.

## 🛠️ Tech Stack
- **Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/), [GSAP](https://greensock.com/gsap/)
- **3D Elements**: [Three.js](https://threejs.org/) via React Three Fiber
- **Icons**: [Lucide React](https://lucide.dev/)

## 📦 Features
- **Modern UI/UX**: Dark-themed, glassmorphic design system.
- **Scroll Animations**: Smooth transitions and parallax effects using Lenis and Framer Motion.
- **Dynamic Routing**: Clean navigation with React Router.
- **Interactive Cursor**: Custom magnetic cursor for enhanced interaction.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.

## 💻 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/zenbyte-labs-portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🌐 Deployment

### Recommended: Vercel
1. Push your code to a GitHub repository.
2. Log in to [Vercel](https://vercel.com).
3. Select your repository and click **Deploy**.
4. Vercel will automatically detect Vite and handle the build.

### Manual Build
To generate a production-ready build:
```bash
npm run build
```
The output will be in the `dist/` directory.
